## 🎯 Skill làm được gì?

### ✨ **Tạo Frontend Components**
- React, Vue, HTML/CSS components
- Landing pages, dashboards, web apps
- UI với design quality cao
- Tránh generic AI aesthetics

### 🎨 **Design Features**
- **Typography**: Font đẹp, unique, không dùng Inter/Arial generics
- **Color**: Palette cohesive với CSS variables
- **Motion**: Animations, micro-interactions
- **Layout**: Unexpected, asymmetry, creative grids
- **Details**: Textures, gradients, shadows, borders

### 🚫 **Tránh những gì?**
- ❌ Generic fonts (Inter, Roboto, Arial)
- ❌ Purple gradients on white
- ❌ Cookie-cutter designs
- ❌ Boring, predictable layouts
- ❌ AI-generated aesthetics

## 📝 Tips để có kết quả tốt nhất

1. **Be Specific**: Chỉ rõ aesthetic direction bạn muốn
2. **Provide Context**: Mục đích, target audience, brand vibe
3. **Give Examples**: "Like Apple's website" hoặc "Similar to Linear's design"
4. **Specify Constraints**: Framework, performance, accessibility

### Aesthetic Directions to Try:
- **Brutalist**: Raw, bold, unconventional
- **Minimalist Luxury**: Elegant, refined, expensive
- **Retro-futuristic**: 80s/90s vibe, neon, analog
- **Organic**: Natural, soft, flowing
- **Editorial**: Magazine-like, typography-driven
- **Maximalist**: Chaotic, layered, expressive
- **Industrial**: Utilitarian, raw, functional

## 🐛 Troubleshooting

### Skill không được gợi ý?
1. Kiểm tra: `ls ~/.claude/skills/frontend-design/SKILL.md`
2. Kiểm tra settings: `cat ~/.claude/settings.json`
3. Restart Claude Code IDE

### Muốn disable skill?
Edit `~/.claude/settings.json`:
```json
{
  "skills": {
    "enabled": []
  }
}
```
